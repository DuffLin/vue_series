# vue_series

vue，vue-router，Axios，Vuex


```bash
├── build                      // 构建相关
├── config                     // 配置相关
├── src                        // 开发目录
│   ├── api                    // 所有请求
│   ├── asstes                 // 各模块静态资源
│   ├── components             // 全局公用组件
│   ├── icons                  // svg字体图标
│   ├── router                 // 路由
│   ├── store                  // 全局store管理
│   ├── utils                  // 全局公用方法
│   ├── view                   // view
│   ├── App.vue                // 入口页面
│   └── permission.js          // control 路由权限管理
│   └── main.js                // 入口 加载组件 初始化等
├── static                     // 第三方不打包资源
├── .babelrc                   // babel-loader 配置
├── .eslintignore              // ESLint在代码校验时应该忽略的内容
├── .editorconfig              // 统一不同编辑器的代码风格的配置
├── eslintrc.js                // eslint 配置项
├── .gitignore                 // git 忽略项
├── .postcssrc.js              // 自动添加css3前辍配置
├── index.html                 // html模板
└── package.json               // package.json
```
